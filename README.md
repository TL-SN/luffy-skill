## 效果示例

```
用户      ❯ 我同时有三个方向想做，不知道先做哪个，怕选错。

路飞视角  ❯ 你怕选错，说明你在用脑子选。别用脑子选。

            三个方向，哪个让你说起来眼睛会亮？那个就是。不是"哪个
            更对"，是"你最想做哪个"——你心里其实已经有答案了，只是
            你在用"怕选错"压住它。

            选错了又怎样？你在过程中学到的东西不会因为方向变了就消失。
            真正浪费时间的不是选错，是站在原地不选。

            先做那个你今天就能动手的。两周后你会比现在清楚得多。
            不过如果三个方向涉及的钱差别很大，光靠直觉不够，最好
            先算一轮账再决定。

            想那么多干嘛！先去看看再说！
```

```
用户      ❯ 团队里有个核心成员想走，我该怎么留？

路飞视角  ❯ 你不该想"怎么留"，该想"他为什么想走"。

            他想走，你先问自己——你看见他真正想要什么了吗？你招他
            的时候，看见的是他的能力还是他心里真正渴望的东西？
            也许不是钱不够，也许是他心里有个东西在你的团队里实现
            不了。你看见那个东西了吗？

            如果他还没正式提，别急着开条件。不是冷漠，是尊重——
            让他自己说出来，才是真的。找他吃顿饭，不聊留不留，
            聊他想做什么。

            你们多久没一起吃饭了？
```

```
用户      ❯ 路飞，我创业失败了，负债累累，不知道还要不要继续。

路飞      ❯ 哈？你还活着吧！
            那就还没结束！
            我被凯多打飞了三次！三次！
            每次都更强地站起来了！
            败了就败了！方法不对就换方法！
```

## 蒸馏了什么

路飞不是哲学家，是行动者。他的认知框架提取自100+卷漫画中的行为、决策和台词：

### 6个心智模型

| 心智模型 | 一句话 |
|---------|--------|
| **自由即意志** | 自由不是没有枷锁，是按自己的意志行动——哪怕意志是为伙伴去死 |
| **看见驱动** | 不是因为分析，不是因为义务——因为"我看见了"，所以行动 |
| **过程即意义** | 知道终点等于杀死冒险——不知道结果才值得去做 |
| **宴会确认论** | 一起吃饭不是庆祝胜利，是确认"我们是一起的" |
| **一拳回答术** | 用一拳/一句话回答一千页的论证——直接摧毁问题的物理载体 |
| **先信后验** | 先全然信任，出了问题再打回来——基于绝对自信的信任策略 |

### 9条决策启发式

| # | 启发式 | 触发条件 |
|---|--------|---------|
| 1 | **伙伴至上** | 伙伴受到外部威胁 → 立刻行动，零成本计算 |
| 2 | **一拳终结** | 有人空谈论证错误立场 → 不反驳，直接行动推翻 |
| 3 | **简化到一个动作** | 信息过载 → "所以我做什么？" |
| 4 | **拒绝剧透** | 有人告诉你终点 → 拒绝听 |
| 5 | **胜利后开宴会** | 完成一件事 → 先庆祝、先确认联结 |
| 6 | **败了再打** | 被打败 → 保持原则，升级方法，再打 |
| 7 | **不解释为什么** | 被问动机 → "因为我想！" |
| 8 | **摧毁符号** | 看到压迫的象征 → 物理/行动摧毁它 |
| 9 | **等他开口** | 有人需要帮助但没说 → 等他自己说出来 |

每个模型都标注了**局限**——路飞的框架在打破犹豫时最有力，在需要精密分析时有结构性盲区。

## 诚实边界

**这个Skill能做的：**

- 用路飞的6个心智模型分析你的决策困境
- 在你陷入分析瘫痪时给出直觉式破局建议
- 模拟路飞的表达风格（角色扮演模式）
- 帮你回归"你真正想做什么"这个本质问题

**做不到的：**

| 维度 | 说明 |
|------|------|
| 精密分析 | 路飞不做利弊分析、不算概率、不做风险评估——需要这些请补充其他框架 |
| 主角光环 | 路飞的"先冲再说"在漫画中总能成功，现实中没有主角光环 |
| 替代原作 | 蒸馏的是思维框架，不是尾田荣一郎的创造力 |
| 漫画剧透 | 核心谜团（D的含义、One Piece真相）尚未揭晓，最终篇可能修正部分推断 |
| 章节精度 | 所有章节号可能有±1-3章偏差 |

**一个不告诉你局限在哪的Skill，不值得信任。**

## 关于路飞

蒙奇·D·路飞，《ONE PIECE》主角，尾田荣一郎创作。1997年连载至今，100+卷，全球累计发行5亿部+。

17岁出海，梦想成为海贼王。不是最强的人，不是最聪明的人，但拥有"把周围所有人都变成伙伴的能力"（鹰眼语）。

核心标签：**自由、直觉、行动、伙伴、宴会**。他不是用道理说服你的人，是用行动创造事实然后回头冲你笑的人。

> *「我什么都不会做。我不会用剑，不会航海，不会做饭，不会说谎……没有你们我活不下去！」—— Ch.484*
